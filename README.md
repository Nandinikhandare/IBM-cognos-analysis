This project is a Retail Sales Dashboard created using IBM Cognos Analytics.
It helps analyze sales performance, customer details, and product trends in a simple and visual way.

Dashboard Overview:
-Total Revenue vs Planned Revenue
-Revenue by Product Line (Year-wise)
-Product Line Performance by Year
-Revenue by Education Level
-Revenue by Marital Status
-Marketing Response by Department
-Sales Performance by Region (Map Visualization)

Highlights:
-Total Revenue is higher than Planned Revenue.
-Sales were highest in 2020.
-Married customers generated more revenue.
-Bachelor degree customers contributed the most.
-North America shows strong sales performance.
